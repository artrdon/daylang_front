# О проекте
Веб-приложение для практики разговорной речи на английском языке с использованием Искусственного Интеллекта. 

## Что из себя представляет репозиторий
В этом репозитории хранится исходный код фронтенда моего первого реального проекта, который дошел до прода. В нем реализованы:
- Регистрация
- Аутентификация
- Оплата
- Запросы к LMM
- Запись голоса
- Голосовой ответ нейросети
- Тестовый ответ нейросети
- Капча

## Стек
- JavaScript
- React
- Docker
- Nginx
